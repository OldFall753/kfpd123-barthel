# kfpd123-barthel - 改良Barthel指数评定量表 🏥

在线Barthel评估工具，用于评估患者的日常生活自理能力。

## 功能

1. **问卷评估** - 10项日常生活能力评估，每项单选打分
2. **评分结果** - 自动计算总分，显示各项得分明细和自理能力等级

## 评分等级

| 总分范围 | 等级 |
|---------|------|
| 80-100分 | 基本自理（轻度依赖） |
| 60-79分  | 中度依赖 |
| 40-59分  | 重度依赖 |
| 20-39分  | 极重度依赖 |
| 0-19分   | 完全依赖 |

## 技术

- 纯 HTML + CSS + JavaScript
- 左侧导航栏 + SPA 无刷新切换
- localStorage 数据持久化
- 移动端自适应
