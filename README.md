# kfpd123-barthel - 康复功能评定平台 🏥

综合性康复功能评估工具，包含8个标准化评估量表，左侧导航栏一键切换。

## 包含量表

| 图标 | 名称 | 章节 | 功能 |
|:----:|------|:----:|------|
| 📋 | 改良Barthel指数 | 第14章 | 日常生活活动能力（ADL）评估 |
| 🧠 | 认知功能评定 | 第5章 | 定向、记忆、注意、执行功能 |
| 🗣️ | 言语-语言功能评定 | 第6章 | 口语表达、理解、复述、命名 |
| 👆 | 感觉功能评定 | 第7章 | 浅感觉、深感觉、复合感觉 |
| 🎯 | 肌张力评定（改良Ashworth） | 第8章 | 痉挛程度分级评估 |
| 💪 | 肌力评定（MMT） | 第9章 | Lovett 6级肌力分级 |
| 🦵 | 关节活动度评定 | 第10章 | ROM活动范围评估 |
| ⚖️ | 协调与平衡功能评定 | 第11章 | Berg平衡量表+协调测试 |

## 功能特色

- 左侧导航栏快速切换评估量表
- 每项评估独立计分并保存结果
- 自动计算总分、判定功能等级
- 各项得分明细 + 进度条可视化
- localStorage 数据持久化
- 移动端自适应（汉堡菜单）

## 技术

- 纯 HTML + CSS + JavaScript（单文件）
- SPA 单页应用，页面切换无刷新
- 数据驱动架构，便于扩展
